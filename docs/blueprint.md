# **App Name**: GreenNodes Core: FinOps Dashboard

## Core Features:

- Executive Dashboard KPIs: Interactive scorecards and charts visualizing 'Total Infrastructure Spend,' 'Wasted Cloud Budget,' 'Active Token Balance,' 'Avoided Carbon Emissions,' infrastructure allocation, and a cloud spend/carbon intensity line graph.
- On-Demand Cloud Scan Simulation: An interactive button triggering a simulated cloud scan process with loading states and toast notifications indicating synchronization with external systems.
- GNC Token Vault & Yield Simulator: Display of the user's GreenNodes Vault balance, subscription tier, and an interactive simulator to calculate future token balances based on a 5% active rollover yield.
- Slack Alert Card Mockup: A visual simulation of an enterprise Slack channel displaying a 'Zombie Resource Detected' Block Kit Card, including financial leak and carbon footprint metrics.
- Resource Termination Workflow: Interactive 'Terminate Instance' button within the Slack alert card, demonstrating a confirmation dialog and a dynamic state change to a resolved success message.
- Personalized Audit Blueprint Tool: A lead capture form for a 'Free 48-Hour Cloud Waste Audit' that, upon submission, simulates the generation of a personalized 'ServiceNow Executive Dashboard blueprint' for the user via a confirmation modal.

## Style Guidelines:

- Primary Color: Vibrant neon mint-green (#34D399) to highlight key data, actions, and interactive elements. (HSL: 159, 67%, 52%).
- Background Color: A deep charcoal with a subtle green tint (#192A24), providing a clean, modern dark mode foundation, complementary to user's suggested backgrounds. (HSL: 159, 15%, 10%).
- Accent Color: A soft, cool green (#8FD2AA) used for secondary indicators and supporting visuals, offering subtle contrast. (HSL: 129, 40%, 60%).
- Alert Color: A strong red (#EB3030) specifically for critical 'waste' alerts, ensuring immediate attention. (HSL: 0, 90%, 60%).
- Body and headline font: 'Inter' (sans-serif), for its modern, clean, and objective aesthetic, suitable for a high-performance developer tool interface.
- Use sleek, minimalist, and potentially glowing line icons to complement the ultra-modern dark theme and emphasize data points or interactive elements.
- Implement a clear sidebar navigation layout for seamless switching between the four main application views (Executive Workspace, GNC Token Vault, Slack ChatOps, Lead Capture).
- Incorporate smooth transitions for view changes, data updates, and button interactions, along with subtle glowing micro-interactions on hover or focus to enhance responsiveness.
